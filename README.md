# GamesForAll_Reviewer
O projeto "Games for All" destina-se a impactar um público-alvo amplo e diversificado. Nossos usuários são jogadores de todos os perfis, abrangendo desde o jogador casual, que busca uma opinião rápida para saber se "vale a pena" adquirir um título, até o jogador mais técnico, focado em performance e requisitos de hardware.
